# 皇冠赛评分规则手册

## 评分计算流程

1. 每个指标按阈值或扣分规则计算得到 0-100 的 threshold_score
2. 维度内指标按 indicator_weight 加权求和，再乘以 dimension_weight 得到维度得分
3. 所有维度得分求和，减去所有扣分项，得到原始岗位分
4. 根据增长率排名，线性映射到 60-100（增长排名分）
5. 若原始分 >= 85，取 max(原始分, 增长排名分)；否则取增长排名分
6. 最终分 = 岗位分 + 组织分（上限 25 分），在岗位组内排名，按 271 分布

## 阈值评分规则

对于 threshold 类型指标：
- raw_value >= threshold_100 → 100 分
- raw_value >= threshold_60 → 60 + (raw_value - threshold_60) / (threshold_100 - threshold_60) * 40
- raw_value < threshold_60 → 0 分

## 扣分规则

对于 deduction 类型指标：
- 扣分 = min(raw_value * deduction_per_unit / deduction_divisor, deduction_cap)
- 扣分项的 dimension_weight 和 indicator_weight 均为 0，独立从总分中扣除
